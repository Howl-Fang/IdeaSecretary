## 📚 什么是树形视图？

想象您的想法像这样：

```
📚 My Knowledge Base
├── 🧠 AI and Machine Learning
│   ├── Deep Learning Basics
│   └── Neural Networks
├── 🎯 Career Planning
│   ├── Short-term Goals
│   └── Long-term Vision
└── 💡 Side Projects
    ├── Mobile App
    └── Web Platform
```

**树形视图就是展示这个结构！**

- ✨ 点击 ▶ 展开/折叠
- 📊 看每个想法有多少字
- 🏷️ 看想法的标签
- ⚡ 快速编辑/删除

---

## 🎨 什么是思维导图？

想象您的想法像这样：

```
        [Main Topic]
            /  |  \
          /    |    \
       Child1 Child2 Child3
        /        |        \
      ...      ...       ...
```

**思维导图就是用图形连接展示！**

- 🎯 节点代表想法
- 〰️ 线条代表关系
- 🔍 悬停看详情
- 🖱️ 右键拖拽浏览
